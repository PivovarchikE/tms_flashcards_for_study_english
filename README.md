# 🇬🇧 English Flashcards API

**English Flashcards API** — это асинхронное веб-приложение на базе **FastAPI**, предназначенное для создания и управления карточками для изучения английских слов. Проект использует асинхронную библиотеку **aiosqlite** для работы с базой данных SQLite.

---

## 📖 Документация и Swagger

Одной из ключевых особенностей проекта является автоматическая генерация интерактивной документации. После запуска сервера вы можете протестировать все функции API прямо в браузере:

* **Swagger UI**: [http://localhost:8000/docs](http://localhost:8000/docs) — интерактивный интерфейс для отправки запросов и просмотра схем данных.
* **ReDoc**: [http://localhost:8000/redoc](http://localhost:8000/redoc) — альтернативный формат документации в виде структурированного справочника.

Каждый эндпоинт снабжен описанием, примерами (examples) и валидацией входящих данных через Pydantic.

---

## 📂 Структура проекта

Проект организован в виде модульной структуры для удобства поддержки:

* **`main.py`** — точка входа в приложение. Содержит настройки FastAPI, логику жизненного цикла (lifespan) и описание всех маршрутов (endpoints).
* **`database.py`** — модуль для работы с БД. Отвечает за создание таблиц, инициализацию начальных данных (seed) и выполнение асинхронных SQL-запросов.
* **`models.py`** — описание схем данных с использованием Pydantic. Здесь определены структуры для создания, обновления и отображения карточек, а также форматы ответов со статистикой.
* **`requirements.txt`** — список зависимостей, необходимых для работы проекта (FastAPI, Uvicorn, aiosqlite и др.).
* **`.gitignore`** — конфигурация для Git, исключающая системные файлы, кэш Python (`__pycache__`), виртуальные окружения и локальную базу данных `flashcards.db`.

## 🔧 Основные API Эндпоинты

Приложение разделено на логические теги для удобства навигации в Swagger:

### Карточки (CRUD)
| Метод | Путь | Описание |
| :--- | :--- | :--- |
| `GET` | `/cards` | Получить список всех карточек с фильтрацией по тексту. |
| `POST` | `/cards` | Создать новую карточку (английский, русский, категория). |
| `GET` | `/cards/{card_id}` | Получить детальную информацию о конкретной карточке. |
| `PUT` | `/cards/{card_id}` | Обновить данные существующей карточки. |
| `DELETE` | `/cards/{card_id}` | Удалить карточку из базы данных. |

### Тренировка и Статистика
| Метод | Путь | Описание |
| :--- | :--- | :--- |
| `GET` | `/cards/random` | Получить случайную карточку для проверки знаний. |
| `GET` | `/stats` | Показать общее количество слов и распределение по категориям. |
